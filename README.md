# IT Ticketing Project

## Overview

This project simulates real-world Help Desk and IT Support tickets using common issues encountered in business environments.

The goal is to demonstrate troubleshooting methodology, ticket documentation, customer communication, issue resolution, and escalation procedures.

## Ticket Categories

- Password Resets
- Account Lockouts
- VPN Issues
- Outlook Issues
- Microsoft Teams Issues
- Printer Issues
- Network Connectivity Issues
- MFA Problems
- Shared Drive Access
- Software Installation Requests

## Skills Demonstrated

- Technical Troubleshooting
- Ticket Documentation
- Customer Support
- Escalation Procedures
- Root Cause Analysis
- Incident Resolution
- Microsoft 365 Support
- Windows 10/11 Support

# Ticket 001 - VPN Connection Failure

## User Issue

User unable to connect to company VPN while working remotely.

## Priority

Medium

## Troubleshooting

1. Verified internet connectivity.
2. Confirmed VPN credentials were correct.
3. Checked DNS resolution.
4. Restarted VPN client.
5. Reviewed VPN configuration settings.

## Resolution

VPN profile contained incorrect server address.

Updated configuration and successfully restored VPN access.

## Escalation

Not required.

## Status

Resolved
